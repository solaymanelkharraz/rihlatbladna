# 🌍 RihlatBladna

Welcome to **RihlatBladna**, a premium SaaS platform designed to bridge the gap between travel agencies and enthusiastic travelers. This modern web application serves as a comprehensive discovery and management tool, featuring an interactive community, advanced booking systems, and dedicated dashboards for both travelers and agencies.

## ✨ Key Features

### For Travelers 🎒
- **Tour Discovery:** Seamlessly search and explore travel offers and detailed itineraries.
- **Agency Directory:** Browse and connect with trusted travel agencies.
- **Community Feed:** Engage with a vibrant social feed to share experiences and travel tips.
- **Personalized Profile:** Manage bookings, preferences, and travel history.

### For Travel Agencies 🏢
- **Dedicated Dashboard:** A centralized hub to oversee operations and analytics.
- **Offer Management:** Create, publish, and manage travel packages (My Offers).
- **Booking Management:** Track and process customer reservations efficiently.
- **Direct Messaging:** Communicate with travelers seamlessly through an integrated messaging system.

## 🚀 Technology Stack

Built with a focus on performance, modern design, and robust architecture:
- **Frontend Framework:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/) for blazing-fast development and optimized production builds.
- **Routing:** [React Router v7](https://reactrouter.com/) for seamless, client-side navigation.
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) for dynamic, utility-first, and responsive UI designs.
- **Networking:** [Axios](https://axios-http.com/) for efficient API requests.
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/) for high-quality, customizable iconography.

## 🛠️ Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/solaymanelkharraz/rihlatbladna.git
   cd rihlatbladna
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173` to see the application in action.

## 📜 Scripts

- `npm run dev`: Starts the Vite development server.
- `npm run build`: Builds the application for production.
- `npm run lint`: Lints the codebase using ESLint.
- `npm run preview`: Locally preview the production build.
